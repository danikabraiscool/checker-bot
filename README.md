# Discord & Roblox OAuth2 Verification App

Прозрачный инструмент OAuth2-верификации пользователей через Discord и Roblox. Отлично подходит для управления доступом и проверки участников в игровых сообществах, ролевых проектах (Roleplay) и фракционных группах.

## 🚀 Особенности
* **Discord OAuth2:** Безопасное получение базовой информации профиля (ID, Username) и списка серверов пользователя.
* **Roblox OAuth2:** Интеграция через OpenID Connect для подтверждения владения аккаунтом Roblox (получение ID, никнейма и ссылки на профиль).
* **Webhook Logging:** Моментальная отправка результатов авторизации напрямую в приватный канал Discord через вебхук (данные не сохраняются на сервере).
* **Прозрачность:** Пользователь видит всю передаваемую информацию прямо на экране успешной авторизации.

## 🛠 Установка и запуск
1. Клонируйте репозиторий.
2. Установите зависимости: `pip install -r requirements.txt`
3. Укажите переменные окружения (Environment Variables):
   * `CLIENT_ID`, `CLIENT_SECRET`, `REDIRECT_URI` (для Discord)
   * `ROBLOX_CLIENT_ID`, `ROBLOX_CLIENT_SECRET`, `ROBLOX_REDIRECT_URI` (для Roblox)
   * `DISCORD_WEBHOOK_URL` (для получения логов)
4. Запустите приложение: `python app.py` (или используйте Gunicorn для облачного хостинга).
